ICON DESIGN
-----------
Shape:  56×56 square, rx=13 (rounded corners)
Fill:   Linear gradient — #2A5298 (top) → #1B3A6B (bottom)
EKG:    Proper PQRST medical waveform
        flat → P-wave bump → Q dip → tall R spike → S dip → T wave → flat
Stroke: 2.5px white, stroke-linecap="round", stroke-linejoin="round"

WORDMARK
--------
Font:     Inter 800 (Extra Bold)
Color:    #1B3A6B (primary) / #FFFFFF (on dark)
Spacing:  letter-spacing -0.8px

COLOR PALETTE
-------------
#1B3A6B   Navy · Primary · Icon, wordmark, headings
#4A9EFF   Blue · Accent · CTAs, links, highlights
#0A1628   Dark Navy · Hero backgrounds
#F7F9FC   Light · Section backgrounds, cards
#FFFFFF   White · Text on dark, icon EKG lines

CLEAR SPACE
-----------
Minimum clear space: equal to the height of the "P" letterform on all sides.
Minimum size: 100px wide (horizontal lockup), 24px (icon only).

DO NOT
------
- Stretch or distort the logo
- Change the font or weights
- Use on backgrounds that reduce contrast below 4.5:1
- Add shadows, outlines, or effects
- Recreate the EKG path freehand
